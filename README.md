## 技术栈

`Vue2 + element-ui`

## 文件夹/文件命名规范

目录的命名规范

1. 目录名全部使用小写,`kebab-case`形式命名,如果需要有多个单词表达,使用中划线连接。如`new-page`、`components`。

文件的命名规范

1. 文件名全部使用小写,`kebab-case`形式命名。如`td-layout.vue`、 `td-header.jsx`。

## 开发约定
